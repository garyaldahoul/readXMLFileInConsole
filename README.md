# Working with XML using .NET & React:


This repository contains 6 different solutions implemented using .NET & React for Reading Data From XML file.Each solution is pushed to separate pull requests to showcase different ways of running the solutions.

1. [Show Target in Console](https://github.com/garyaldahoul/readXMLFileInConsole/tree/master)
2. [Enter ID in Console](https://github.com/garyaldahoul/readXMLFileInConsole/tree/XML-Console-EnterID)
3. [Console Write Data to SQL DB](https://github.com/garyaldahoul/readXMLFileInConsole/tree/XML-Console-DB)
4. [Show Target .NET MVC](https://github.com/garyaldahoul/readXMLFileInConsole/tree/XML-MVC)
5. [MVC Write Data to SQL DB](https://github.com/garyaldahoul/readXMLFileInConsole/tree/XML-MVC-DB)
6. [Display Data on FE-React & .NET Server REST API](https://github.com/garyaldahoul/readXMLFileInConsole/tree/XML-API-DB)
